Design #1

2/6/2019

Where we are

  • Data are processed and loaded
  • I've developed a basic chart template and layout

Next steps

  • Feedback on chart template
  • Refine language to describe the values in the chart. To be clear, it's log(average change in pop per decade)? What are the best labels for the chart axes?
  • Develop the story. How many of these chart panels do we need? What do they say?

Notes on the chart

  • There is a lot of dot overlap if we plot dots individually. That motivated the heatmap format below. If you really want to see the dots, click here Â»
  • There are a lot of parameters to filter on. In general, I'd advocate for simplicity and I'd only filter by variables that are discussed in the text. I recommend not having user-selectable filters as I don't think the time required to enable this would pass a cost-benefit test. I think the value here is in curating views that you have researched and then explaining those views to the reader.

Next steps

  • Enable place search / highlighting to facilitate narratives about specific places
